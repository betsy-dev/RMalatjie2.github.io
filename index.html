Campus Counseling App design

Campus Student Counseling Booking System: A collaborative project aimed at streamlining the booking process for student counseling services on campus. The prototype was developed using NetBeans, focusing on enhancing student accessibility to mental health resources.

As the front-end designer, I was responsible for creating user-friendly forms and interfaces that facilitate smooth interactions for students seeking counseling services.

I designed intuitive forms that enhance user experience, enabling students to navigate the booking process effortlessly. A cohesive visual design was implemented across all forms, utilizing consistent color schemes, typography, and layout principles for an aesthetically pleasing interface. The project was developed using NetBeans, with images and visual assets created in Canva
